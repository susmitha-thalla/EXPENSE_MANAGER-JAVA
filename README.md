# Expense Manager (Java Console Application)

A **Java console-based Expense Manager** that helps users record, store, and analyze their expenses.
The application follows a **clean layered architecture** using packages such as `app`, `model`, `service`, and `util`.

---

##  Features

* Add and store expenses
* Track monthly expenses
* Generate expense reports
* File-based expense storage
* Organized package structure using **OOP principles**

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling
* Java Collections Framework
* Command Line Interface (CLI)

## How to Run

### 1. Clone the repository

```
git clone https://github.com/susmitha-thalla/expense-manager-java.git
```

### 2. Navigate to the project folder

```
cd expense-manager-java
```

### 3. Compile the project

```
javac src/app/ExpenseApp.java
```

### 4. Run the application

```
java src.app.ExpenseApp
```
